# Data Structures and Algorithm Practice

## Java, Python, & C++

### Usage

> Java

- Open in IntelliJ

> Python

- Make a virtual environment
- Install the packages using poetry or requirements.txt (use Makefile to generate it)

> C++

- Open in CLion
- use CMakeLists.txt
